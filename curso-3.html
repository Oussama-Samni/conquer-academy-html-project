Frontend Developer Bootcamp

6 meses · 400+ horas Online en vivo 2.999 EUR

Descripción del programa

El Frontend Developer Bootcamp te enseñará a crear interfaces web modernas, accesibles y de alto rendimiento. En 6 meses intensivos, pasarás de conocer los fundamentos de HTML y CSS a dominar React y Next.js como un profesional.

Aprenderás a pensar en términos de componentes, a manejar el estado de las aplicaciones y a optimizar la experiencia de usuario en cualquier dispositivo.

Al finalizar, tendrás un portfolio sólido con proyectos reales y estarás listo para trabajar como frontend developer en cualquier empresa de tecnología.

Por que elegir este bootcamp

  • Garantía de empleo: Si no consigues trabajo en 6 meses, te devolvemos el dinero
  • Accesibilidad web: Aprendes WCAG y ARIA desde el principio
  • Diseño incluido: Nociones de UI/UX para trabajar con diseñadores
  • Performance first: Optimización de carga y Core Web Vitals
  • Proyectos reales: Portfolio con aplicaciones de nivel profesional
  • Mentoría 1:1: Sesiones personalizadas con instructores expertos

Temario completo

Módulo 1: HTML y CSS Profesional (4 semanas)

Los cimientos de cualquier interfaz web.

  • HTML5 semántico y accesibilidad (WCAG, ARIA)
  • CSS3 moderno: Custom Properties, Cascade Layers
  • Flexbox y CSS Grid a fondo
  • Responsive design y mobile-first
  • Animaciones y transiciones CSS
  • Proyecto: Landing page responsive accesible
Módulo 2: JavaScript para el Frontend (5 semanas)

El lenguaje que da vida a las interfaces.

  • JavaScript ES6+: destructuring, spread, modules
  • Manipulación del DOM y eventos
  • Programación asíncrona: Promises, async/await
  • Fetch API y consumo de REST APIs
  • Introducción a TypeScript
  • Proyecto: App de clima con geolocalización
Módulo 3: React Fundamentos (5 semanas)

El framework mas demandado para construir interfaces.

  • Componentes, props y estado
  • Hooks: useState, useEffect, useContext
  • React Router y navegación SPA
  • Formularios y validación
  • Consumo de APIs desde React
  • Proyecto: App de gestión de tareas con React
Módulo 4: React Avanzado y Next.js (5 semanas)

Lleva tus aplicaciones al siguiente nivel.

  • Estado global con Context API y Zustand
  • Custom hooks y patrones avanzados
  • Next.js: SSR, SSG, App Router
  • Tailwind CSS para estilos rápidos
  • Optimización de rendimiento (lazy loading, memoization)
  • Proyecto: Blog con Next.js y CMS headless
Módulo 5: Testing y Calidad (3 semanas)

Escribe código que funciona y se puede mantener.

  • Testing con Jest y React Testing Library
  • E2E con Cypress
  • Accesibilidad automatizada con axe
  • Code reviews y buenas prácticas
  • Proyecto: Suite de tests para el proyecto anterior
Módulo 6: Proyecto Final y Empleabilidad (4 semanas)

Construye tu proyecto estrella y prepara tu entrada al mercado laboral.

  • Proyecto frontend completo con Next.js
  • Despliegue en Vercel y optimización
  • Optimización de CV y perfil de LinkedIn
  • Mock interviews técnicas
  • Proyecto: Portfolio profesional + app final

Preguntas frecuentes

Es necesario saber diseñar para ser frontend developer?

No. El frontend developer traduce diseños a código; no los crea desde cero. Te enseñamos a interpretar y colaborar con diseñadores, pero no necesitas saber Figma o Photoshop.

Cual es la diferencia con el Fullstack Bootcamp?

El Frontend Bootcamp se especializa exclusivamente en interfaces: HTML, CSS, JavaScript y React. No incluye backend ni bases de datos. Es ideal si quieres especializarte en la parte visual e interactiva de las aplicaciones.

Aprenderemos React o Vue o Angular?

Nos centramos en React, que es el framework mas demandado del mercado. Una vez que dominas React, aprender Vue o Angular es mucho mas fácil — los conceptos son similares.

Que salario puedo esperar al terminar?

Nuestros egresados trabajan como Frontend Developer con salarios de entrada entre 25.000 y 38.000 EUR/año en Espana. Los perfiles con experiencia en Next.js y TypeScript suelen estar en el rango alto.
