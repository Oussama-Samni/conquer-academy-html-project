Descripcion del programa

El Data Engineering Bootcamp te ensenara a disenar, construir y mantener la infraestructura de datos que alimenta las decisiones de negocio. Es uno de los perfiles mas demandados y mejor pagados del mercado tecnologico.

Aprenderas a trabajar con grandes volumenes de datos, a construir pipelines ETL/ELT robustos y a orquestar flujos de datos complejos en entornos cloud.

Al finalizar, domineras las herramientas y tecnologias que usan empresas como Spotify, Airbnb o Netflix para gestionar sus datos a escala.

Temario completo

Modulo 1: Python y SQL para Datos (5 semanas)

Los lenguajes imprescindibles para cualquier data engineer.

  • Python avanzado: funciones, clases, modulos
  • Pandas y NumPy para manipulacion de datos
  • SQL avanzado: CTEs, window functions, optimizacion
  • PostgreSQL y DuckDB
  • Git y entornos virtuales
  • Proyecto: Pipeline de analisis de datos con Python y SQL
Modulo 2: ETL y Data Pipelines (5 semanas)

Construye pipelines de datos robustos y escalables.

  • Conceptos ETL vs ELT
  • Apache Airflow: DAGs, operators, scheduling
  • dbt: transformaciones SQL versionadas
  • Data quality y testing de pipelines
  • Manejo de errores y reintentos
  • Proyecto: Pipeline ETL con Airflow y dbt
Modulo 3: Data Warehousing (4 semanas)

Diseña y gestiona almacenes de datos empresariales.

  • Arquitecturas: Data Warehouse, Data Lake, Lakehouse
  • Snowflake: virtual warehouses, stages, streams
  • BigQuery: particionado, clustering, optimizacion
  • Modelado dimensional: star schema, snowflake schema
  • Proyecto: Data Warehouse en Snowflake con dbt
Modulo 4: Big Data con Apache Spark (5 semanas)

Procesa grandes volumenes de datos con el estandar de la industria.

  • Apache Spark: RDDs, DataFrames, Datasets
  • PySpark para procesamiento distribuido
  • Spark SQL y optimizacion de queries
  • Delta Lake para datos transaccionales
  • Spark Streaming: procesamiento en tiempo real
  • Proyecto: Procesamiento de logs a escala con Spark
Modulo 5: Streaming con Kafka (4 semanas)

Construye arquitecturas de datos en tiempo real.

  • Apache Kafka: topics, particiones, consumer groups
  • Kafka Streams y ksqlDB
  • Patrones: event sourcing, CQRS
  • Integracion Kafka + Spark Streaming
  • Proyecto: Pipeline de eventos en tiempo real
Modulo 6: Cloud Data Platforms (5 semanas)

Despliega y gestiona infraestructura de datos en la nube.

  • AWS: S3, Glue, Redshift, Athena, EMR
  • GCP: BigQuery, Dataflow, Pub/Sub, Composer
  • Azure: Synapse Analytics, Data Factory, Event Hubs
  • Infraestructura como codigo con Terraform
  • Proyecto: Arquitectura multi-cloud de datos
Modulo 7: Proyecto Final y Empleabilidad (5 semanas)

Construye una plataforma de datos completa de nivel empresarial.

  • Arquitectura end-to-end de datos
  • DataOps: CI/CD para pipelines de datos
  • Documentacion y data catalog
  • Preparacion de entrevistas tecnicas
  • Proyecto: Plataforma de datos completa

Preguntas frecuentes

Cual es la diferencia entre Data Engineer y Data Scientist?

El Data Engineer construye y mantiene la infraestructura de datos: pipelines, almacenes, plataformas. El Data Scientist usa esos datos para crear modelos de ML y analisis. Son roles complementarios — los data scientists dependen del trabajo de los data engineers.

Necesito saber matematicas o estadistica?

No en profundidad. El data engineering es mas ingeniero que cientifico. Necesitas logica, SQL solido y Python. Las matematicas avanzadas son mas propias del data science.

Que salario puedo esperar al terminar?

Los data engineers tienen salarios de entrada entre 35.000 y 50.000 EUR/ano en Espana, con rapida progresion. Es uno de los perfiles mejor pagados del sector tech por la alta demanda y poca oferta de profesionales.

El bootcamp cubre machine learning?

No directamente. El bootcamp se centra en ingenieria de datos: pipelines, almacenamiento y procesamiento. Sin embargo, aprenderas a construir la infraestructura que los modelos de ML necesitan para funcionar en produccion.
