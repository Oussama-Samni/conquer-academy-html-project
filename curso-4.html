Descripción del programa

El Data Engineering Bootcamp te enseñará a diseñar, construir y mantener la infraestructura de datos que alimenta las decisiónes de negocio. Es uno de los perfiles mas demandados y mejor pagados del mercado tecnológico.

Aprenderás a trabajar con grandes volúmenes de datos, a construir pipelines ETL/ELT robustos y a orquestar flujos de datos complejos en entornos cloud.

Al finalizar, dominarás las herramientas y tecnologías que usan empresas como Spotify, Airbnb o Netflix para gestionar sus datos a escala.

Temario completo

Módulo 1: Python y SQL para Datos (5 semanas)

Los lenguajes imprescindibles para cualquier data engineer.

  • Python avanzado: funciones, clases, módulos
  • Pandas y NumPy para manipulación de datos
  • SQL avanzado: CTEs, window functions, optimización
  • PostgreSQL y DuckDB
  • Git y entornos virtuales
  • Proyecto: Pipeline de análisis de datos con Python y SQL
Módulo 2: ETL y Data Pipelines (5 semanas)

Construye pipelines de datos robustos y escalables.

  • Conceptos ETL vs ELT
  • Apache Airflow: DAGs, operators, scheduling
  • dbt: transformaciones SQL versionadas
  • Data quality y testing de pipelines
  • Manejo de errores y reintentos
  • Proyecto: Pipeline ETL con Airflow y dbt
Módulo 3: Data Warehousing (4 semanas)

Diseña y gestiona almacenes de datos empresariales.

  • Arquitecturas: Data Warehouse, Data Lake, Lakehouse
  • Snowflake: virtual warehouses, stages, streams
  • BigQuery: particionado, clustering, optimización
  • Modelado dimensional: star schema, snowflake schema
  • Proyecto: Data Warehouse en Snowflake con dbt
Módulo 4: Big Data con Apache Spark (5 semanas)

Procesa grandes volúmenes de datos con el estándar de la industria.

  • Apache Spark: RDDs, DataFrames, Datasets
  • PySpark para procesamiento distribuido
  • Spark SQL y optimización de queries
  • Delta Lake para datos transaccionales
  • Spark Streaming: procesamiento en tiempo real
  • Proyecto: Procesamiento de logs a escala con Spark
Módulo 5: Streaming con Kafka (4 semanas)

Construye arquitecturas de datos en tiempo real.

  • Apache Kafka: topics, particiones, consumer groups
  • Kafka Streams y ksqlDB
  • Patrones: event sourcing, CQRS
  • Integración Kafka + Spark Streaming
  • Proyecto: Pipeline de eventos en tiempo real
Módulo 6: Cloud Data Platforms (5 semanas)

Despliega y gestiona infraestructura de datos en la nube.

  • AWS: S3, Glue, Redshift, Athena, EMR
  • GCP: BigQuery, Dataflow, Pub/Sub, Composer
  • Azure: Synapse Analytics, Data Factory, Event Hubs
  • Infraestructura como código con Terraform
  • Proyecto: Arquitectura multi-cloud de datos
Módulo 7: Proyecto Final y Empleabilidad (5 semanas)

Construye una plataforma de datos completa de nivel empresarial.

  • Arquitectura end-to-end de datos
  • DataOps: CI/CD para pipelines de datos
  • Documentación y data catalog
  • Preparación de entrevistas técnicas
  • Proyecto: Plataforma de datos completa

Preguntas frecuentes

Cual es la diferencia entre Data Engineer y Data Scientist?

El Data Engineer construye y mantiene la infraestructura de datos: pipelines, almacenes, plataformas. El Data Scientist usa esos datos para crear modelos de ML y análisis. Son roles complementarios — los data scientists dependen del trabajo de los data engineers.

Necesito saber matemáticas o estadística?

No en profundidad. El data engineering es mas ingeniero que científico. Necesitas lógica, SQL sólido y Python. Las matemáticas avanzadas son mas propias del data science.

Que salario puedo esperar al terminar?

Los data engineers tienen salarios de entrada entre 35.000 y 50.000 EUR/año en Espana, con rápida progresión. Es uno de los perfiles mejor pagados del sector tech por la alta demanda y poca oferta de profesionales.

El bootcamp cubre machine learning?

No directamente. El bootcamp se centra en ingeniería de datos: pipelines, almacenamiento y procesamiento. Sin embargo, aprenderás a construir la infraestructura que los modelos de ML necesitan para funcionar en producción.
