Descripcion del programa

El Fullstack Developer Bootcamp es nuestro programa mas completo y demandado. En 12 meses intensivos, te transformaras de principiante a desarrollador fullstack profesional, capaz de construir aplicaciones web completas desde cero.

Aprenderas a dominar tanto el frontend (lo que ve el usuario) como el backend (la logica del servidor), bases de datos, y las herramientas de DevOps necesarias para desplegar y mantener aplicaciones en produccion.

Al finalizar, tendras un portfolio con 8+ proyectos reales y estaras preparado para trabajar como desarrollador fullstack en cualquier empresa tecnologica.

  • Clases en vivo: Aprende en tiempo real con instructores expertos

Temario completo

Modulo 1: Fundamentos Web (4 semanas)

Construye una base solida en las tecnologias fundamentales de la web.

  • HTML5 semantico y accesibilidad
  • CSS3: Flexbox, Grid, responsive design
  • Git y GitHub: control de versiones
  • Linea de comandos y herramientas de desarrollo
  • Proyecto: Portfolio personal responsive
Modulo 2: JavaScript Moderno (6 semanas)

Domina el lenguaje de programacion mas importante de la web.

  • JavaScript ES6+: variables, funciones, arrays, objetos
  • Programacion asincrona: Promises, async/await
  • Manipulacion del DOM y eventos
  • Fetch API y consumo de APIs REST
  • Introduccion a TypeScript
  • Proyecto: Aplicacion de gestion de tareas
Modulo 3: React y Frontend Avanzado (8 semanas)

Aprende el framework frontend mas demandado del mercado.

  • React: componentes, props, state, hooks
  • React Router y navegacion SPA
  • Estado global con Context API y Redux
  • Testing con Jest y React Testing Library
  • Proyecto: Clon de e-commerce (frontend)
Modulo 4: Node.js y Backend (8 semanas)

Construye servidores robustos y APIs profesionales.

  • Node.js: fundamentos y modulos
  • Express.js: routing, middleware, MVC
  • APIs REST: diseno, documentacion, versionado
  • Autenticacion: JWT, OAuth, sesiones
  • Proyecto: API REST completa para el e-commerce
Modulo 5: Bases de Datos (6 semanas)

Aprende a disenar y gestionar bases de datos relacionales y NoSQL.

  • SQL y PostgreSQL: queries, joins, indices
  • MongoDB: documentos, agregaciones, indices
  • ORMs: Prisma, Mongoose
  • Redis: cache y sesiones
  • Proyecto: Integracion de base de datos en el e-commerce
Modulo 6: DevOps y Cloud (4 semanas)

Despliega y mantiene aplicaciones en produccion.

  • Docker: contenedores, imagenes, docker-compose
  • CI/CD: GitHub Actions, automatizacion
  • AWS: EC2, S3, RDS, Lambda
  • Proyecto: Despliegue del e-commerce en AWS
Modulo 7: Proyecto Final (8 semanas)

Desarrolla una aplicacion completa de nivel profesional.

  • Planificacion y arquitectura del proyecto
  • Desarrollo en equipo con metodologias agiles
  • Presentacion ante empresas colaboradoras
  • Proyecto: Aplicacion fullstack completa (a elegir)
Modulo 8: Preparacion Laboral (4 semanas)
  • Optimizacion de CV y LinkedIn
  • Preparacion de entrevistas tecnicas
  • Practicas de algoritmos y estructuras de datos
  • Sesiones de mock interviews con recruiters

Si, el programa esta disenado para personas que trabajan. Las clases en vivo son por la tarde/noche y los fines de semana. Necesitaras dedicar unas 20-25 horas semanales.

Ofrecemos garantia de empleo. Si no consigues trabajo en 6 meses tras graduarte, te devolvemos el 100% del importe pagado.

Absolutamente. El bootcamp esta disenado para principiantes. Empezamos desde cero y te acompanamos en cada paso.

Nuestros egresados trabajan como: Fullstack Developer, Frontend Developer, Backend Developer, Software Engineer. El salario medio de entrada es de 28.000-35.000 EUR/ano.
