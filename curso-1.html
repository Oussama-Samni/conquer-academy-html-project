Descripción del programa

El Fullstack Developer Bootcamp es nuestro programa mas completo y demandado. En 12 meses intensivos, te transformarás de principiante a desarrollador fullstack profesional, capaz de construir aplicaciones web completas desde cero.

Aprenderás a dominar tanto el frontend (lo que ve el usuario) como el backend (la lógica del servidor), bases de datos, y las herramientas de DevOps necesarias para desplegar y mantener aplicaciones en producción.

Al finalizar, tendrás un portfolio con 8+ proyectos reales y estaras preparado para trabajar como desarrollador fullstack en cualquier empresa tecnológica.

  • Clases en vivo: Aprende en tiempo real con instructores expertos

Temario completo

Módulo 1: Fundamentos Web (4 semanas)

Construye una base sólida en las tecnologías fundamentales de la web.

  • HTML5 semántico y accesibilidad
  • CSS3: Flexbox, Grid, responsive design
  • Git y GitHub: control de versiones
  • Línea de comandos y herramientas de desarrollo
  • Proyecto: Portfolio personal responsive
Módulo 2: JavaScript Moderno (6 semanas)

Domina el lenguaje de programación mas importante de la web.

  • JavaScript ES6+: variables, funciones, arrays, objetos
  • Programación asíncrona: Promises, async/await
  • Manipulación del DOM y eventos
  • Fetch API y consumo de APIs REST
  • Introducción a TypeScript
  • Proyecto: Aplicación de gestión de tareas
Módulo 3: React y Frontend Avanzado (8 semanas)

Aprende el framework frontend mas demandado del mercado.

  • React: componentes, props, state, hooks
  • React Router y navegación SPA
  • Estado global con Context API y Redux
  • Testing con Jest y React Testing Library
  • Proyecto: Clon de e-commerce (frontend)
Módulo 4: Node.js y Backend (8 semanas)

Construye servidores robustos y APIs profesionales.

  • Node.js: fundamentos y módulos
  • Express.js: routing, middleware, MVC
  • APIs REST: diseño, documentación, versionado
  • Autenticación: JWT, OAuth, sesiones
  • Proyecto: API REST completa para el e-commerce
Módulo 5: Bases de Datos (6 semanas)

Aprende a diseñar y gestionar bases de datos relacionales y NoSQL.

  • SQL y PostgreSQL: queries, joins, índices
  • MongoDB: documentos, agregaciones, índices
  • ORMs: Prisma, Mongoose
  • Redis: caché y sesiones
  • Proyecto: Integración de base de datos en el e-commerce
Módulo 6: DevOps y Cloud (4 semanas)

Despliega y mantiene aplicaciones en producción.

  • Docker: contenedores, imágenes, docker-compose
  • CI/CD: GitHub Actions, automatización
  • AWS: EC2, S3, RDS, Lambda
  • Proyecto: Despliegue del e-commerce en AWS
Módulo 7: Proyecto Final (8 semanas)

Desarrolla una aplicación completa de nivel profesional.

  • Planificación y arquitectura del proyecto
  • Desarrollo en equipo con metodologías ágiles
  • Presentación ante empresas colaboradoras
  • Proyecto: Aplicación fullstack completa (a elegir)
Módulo 8: Preparación Laboral (4 semanas)
  • Optimización de CV y LinkedIn
  • Preparación de entrevistas técnicas
  • Practicas de algoritmos y estructuras de datos
  • Sesiones de mock interviews con recruiters
